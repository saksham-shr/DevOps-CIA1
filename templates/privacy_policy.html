Privacy Policy

1. Restricted Access

This Report Generator system is exclusively for use by authorized faculty members of the Department of AI, ML & Data Science at Christ (Deemed to be University). Access is restricted to whitelisted email addresses only.

2. Data Collection

We collect the following information:

  • Email addresses for authentication
  • Report data and drafts created by users
  • Device information and login sessions for security monitoring
  • IP addresses for security and audit purposes

3. Data Usage

Your data is used solely for:

  • Generating academic activity reports
  • Maintaining draft reports for your convenience
  • Security monitoring to prevent unauthorized access
  • Audit trails for compliance purposes

4. Data Security

We implement security measures including:

  • Password hashing for secure authentication
  • Device fingerprinting to detect unauthorized access
  • Session tracking for security monitoring
  • Secure file storage for uploaded documents

5. Device Monitoring

For security purposes, we track:

  • Device information and browser details
  • IP addresses
  • Login times and sessions
  • New device logins (administrators are notified)

Important: Sharing your credentials with unauthorized persons is strictly prohibited. New device logins are monitored and may result in account suspension if suspicious activity is detected.

6. Data Retention

Draft reports and user data are retained until deleted by the user or as required by institutional policies.

7. Contact

For privacy concerns, please contact the system administrator.
