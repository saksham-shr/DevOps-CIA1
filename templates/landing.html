About This Software

The Academic Activity Report Generator is a comprehensive web-based system designed to streamline the creation of professional activity reports for the Department of AI, ML & Data Science. This tool enables faculty members to efficiently document academic activities, workshops, seminars, and events with a user-friendly interface.

Key Features:

  • Multi-section form with intuitive navigation
  • Automatic PDF report generation
  • Draft saving and management
  • Digital signature storage and reuse
  • Support for multiple file uploads (images and PDFs)
  • Secure authentication and access control
  • Location tracking for security monitoring
